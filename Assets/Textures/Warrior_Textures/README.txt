Medieval Warrior texture maps split from the generated material sheet.

Unity usage:
- T_Warrior_BaseColor_1024.png -> Base Map / Albedo
- T_Warrior_Normal_1024.png -> Normal Map (set Texture Type to Normal map)
- T_Warrior_Roughness_1024.png -> Smoothness/Roughness reference
- T_Warrior_Metallic_1024.png -> Metallic Map
- T_Warrior_AO_1024.png -> Occlusion Map

Note: These are separated from a generated preview image, so they are concept/reference textures rather than production-ready UV-specific maps.
